Random Gutenburg

My first independent project while learning JavaScript, HTML5, and CSS3 solves the problem teachers and assessment writers face in finding non-copyrighted material to use when crafting English language arts high school assessments. This simple website generates a random book(s), author, or topic from Project Gutenberg to potentially use as a text-based passage for assessing national learning standards. It can also be used to find some new reading material from the public domain. Enjoy Reading!
